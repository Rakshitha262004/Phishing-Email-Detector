# Phishing-Email-Detector
A machine learning-based Phishing Email Detector that classifies emails as phishing or safe. It uses Python and AI models to analyze email content and URLs, providing confidence scores for predictions. Features a web dashboard for real-time detection and easy integration with APIs.


## Technologies Used
- Python
- Scikit-learn (Logistic Regression / ML models)
- Flask / FastAPI for web interface
- HTML, CSS, JavaScript for frontend
- Pandas & NumPy for data preprocessing

---

## Features
- Detect phishing emails using AI/ML
- Analyze email subject, body, and URLs
- Provides confidence score for predictions
- Simple web-based dashboard interface
- Easily extendable with new datasets
